HTML-taggar


<html>

Taggen <html> används för att representera roten av ett HTML dokument. Alla HTML element (förutom <!DOCTYPE>) ska vara inom taggen.

    <!DOCTYPE html>
    <html>
        <head>
            <title>Dokumentets titel</title>
        </head>
        <body>
            <h1>Sidans innehåll</h1>
            <p>Innehåll</p>
        </body>
    </html>


<head>

<head> används för att mata webbläsaren med data om sidan. Detta kan vara allt från CSS-stylesheet, språk och karaktärsuppsättning, till vad som ska stå på fliken när sidan är öppen i en webbläsare.

    <head>
        <!-- Bestämmer karaktärsuppsättning -->
        <meta charset="UTF-8">

        <!-- Kompatibilitet med webbläsare -->
        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 

        <!-- Visuell skalning baserat på bildskärmstorlek -->
        <meta name="viewport" content="width=device-width, 
        initial-scale=1.0" >

        <!-- Sökord för sökmotorer -->
        <meta name="keywords" content="HTML, CSS">

        <!-- Länk till CSS-stylesheet som ska användas -->
        <link rel="stylesheet" type="text/css" 
        href="../mystyle.css">
        
        <!-- Vad som syns i fliken i webbläsaren -->
        <title>Chris' Corner</title>
    </head>


<body>

I <body> defineras sidans innehåll, som rubriker, textstycken, bilder, länkar och så vidare.

    <body>
        <h1>
            Sidans Rubrik
        </h1>
        <p>Sidans innehåll...</p>
        <p>...</p>
    </body>


<footer>

...och som slutet på kroppen så kan man ha en <footer>. Här läggs oftast information om ägarskap, copyright, kontakt, en sidkarta och relaterade dokument.

    <footer>
        <p>Copyright: "xxx"</p>
    </footer>

    <h4>Fjärde storleken <h4></h4>
    <h5>Femte storleken <h5></h5>

Bilder <img>

Om man vill visa en bild så kan man använda sig av <img>. Bilden behöver en fungerande sökväg för att kunna visas.

    <!-- Bilden deklareras här tillsammans med en sökväg, ett id 
        för CSS samt en alternativ text om bilden inte visas -->
    <img src ="../images/scroll-gray.png" 
    class="imgBack" 
    alt="Tillbaka">
                
    <!-- Samma med en trasig länk -->
    <img src ="./images/broken-link.png" 
    class="imgBack" 
    alt="Tillbaka">


Radbrytning <br>

Vill man ha lite utrymme mellan eller inom sina element så kan man använda taggen <br>. Detta används eftersom HTML inte läser av radavstånd i sitt innehåll utan att det deklareras.

    <p>Inget extra mellanrum mellan här och 

            där</p>

    <p>Men här <br> kom en radbrytning.</p>

Listor <ol>, <ul> & <dl>

Det finns tre sätt att göra listor på i HTML, ordnade, oordnade och beskrivande (descriptive) listor.

Ordnade <ol> och oordnade <ul> listor är numrerade respektive punktmarkerade. Siffrorna samt punkterna kan ändras med stilsättning.

Bägge listtyper använder sig av list-föremål (list item) <li> för att populeras

    <ol>
        <li><h3>Ordnad lista <ol></h3><li>
        <li>Första punkten</li>
        <li>Andra punkten</li>
        <li>Tredje punkten</li>
    </ol>
    <ul>
        <li><h3>Oordnad lista <ul></h3><li>
        <li>Första punkten</li>
        <li>Andra punkten</li>
        <li>Tredje punkten</li>
    </ul>

  4. Tredje punkten

Beskrivande listor <dl> används för att para ihop en övertitel <dt> med en underpunkt <dd>

    <dl>
        <dt>Första rubrik</dt>
        <dd>Tillhörande text för första rubriken</dd>
        <dt>Andra rubrik</dt>
        <dd>Tillhörande text för andra rubriken</dd>
        <dd>Ännu en tillhörande text för andra rubriken</dd>
    </dl>

Formulär <form>

HTML kan hantera formulär med element för etiketter <label>, samt inmatningsfunktioner <input> i form av celler för skrivet innehåll <input type="text">, krysslådor "checkbox" och olika knappar så som för att skicka in "submit" och radioknappar "radio". 

    <form>
        <label for="firstrow">Första radens etikett.</label>
        <input type="text" id="firstrow" 
        name="firstrow" value="Första raden"><br>
        
        <label for="secondrow">Andra radens etikett.</label>
        <input type="text" id="secondrow" 
        name="secondrow" value="Andra raden"><br>

        <label for="firstbox">Första kryssrutan</label>
        <input type="checkbox" id="firstbox" 
        name="firstbox" checked><br>

        <label for="secondbox">Andra kryssrutan</label>
        <input type="checkbox" id="secondbox" 
        name="secondbox"><br>

        <label for="firstradio">Första radioknappen</label>
        <input type="radio" id="firstradio" 
        name="radio" value="first"><br>

        <label for="secondradio">Andra radioknappen</label>
        <input type="radio" id="secondradio" 
        name="radio" value="second"><br>

        <input type="submit" value="Skicka">
    </form>








Tabeller <table>

Man kan skapa tabeller direkt i HTML med tillhörande celler och rubriker.

En tabells sidhuvud <thead> och kropp <tbody> deklareras tillsammans med rubrik <th>, antal rader <tr> och celler <td>.

    <table>
        <thead>
            <tr>
                <th colspan="3">Tabellens rubrik</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Tabellens kropp</td>
                <td>med tre</td>
                <td>kolumner</td>
            </tr>
            <tr>
                <td>och</td>
                <td>två</td>
                <td>rader.</td>
            </tr>
        </tbody>
    </table>

<div>

Taggen <div> används för att dela upp en webbsida i flera sektioner som kan ha individuell stilsättning.

Det skrivs ut som <div>

<div> taggen kan stiliseras med CSS och fungerar kapslade i varandra. I exemplet nedan används inline CSS <style="background-color:"> för att sätta bakgrundsfärger.

    <div style="background-color:black">
        <p>Här är taggen satt med svart bakgrund.</p>
        <div style="background-color:grey">
            <p>Här är taggen inkapslad och satt med grå bakgrund.</p>
        </div>
        <p>...och här är lite mer svart firstDiv.</p>
    </div>

<span>

Taggen <span> är en inline container för att markera en del av en text eller dokument.

    <p>
        Här är en vit text men med ett 
        <span style="color:blue">blått</span> ord.
    </p>

Här är en vit text men med ett blått ord.
