CSS-egenskaper

Font

Med egenskapen font kan man ändra typsnittet på en text. CSS har i grunden fem font-familjer. Mer om detta kan man läsa här.

    /* CSS */
    #textfont1 { 
        font-family:"Times New Roman", Times, serif
        }
    #textfont2 { 
        font-family:"Lucida Console", "Courier New", monospace
        }   

    <!-- HTML -->
    <p id=textfont1>Här använder vi ett typsnitt...</p>
    <p id=textfont2>...och här ett annat</p>

Text-align

Med denna egenskap kan vi justera en texts utskrift horizontellt.

    /* CSS */
    #textalignleft {text-align:left;}
    #textaligncenter {text-align:center;}
    #textalignright {text-align:right;}

    <!-- HTML -->
    <p id="textalignleft">Vänster</p>
    <p id="textaligncenter">Mitten</p>
    <p id="textalignright">Höger</p>

Text-decoration

Egenskapen används för att sätta eller ta bort viss dekoration från element.

Man kan ha en linje ovanför, genom eller under en text.

    /* CSS */
    #textdecorover {text-decoration:overline;}
    #textdecorthrough {text-decoration:line-through;}
    #textdecorunder {text-decoration:underline;}
    #textdecornone {text-decoration:none;}

    <!-- HTML -->
    <p id="textdecorover">Över</p>
    <p id="textdecorthrough">Genom</p>
    <p id="textdecorunder">Under</p>
    <a id="textdecornone" href="properties.html">
        Länk utan underlinje.</a>

Word-spacing

Används för att bestämma utrymmet mellan ord i ett element. Standard är 0.25em och det går att sätta ett negativt värde.

    /* CSS */
    #wordspacing1 {word-spacing:1em;}
    #wordspacing2 {word-spacing:2em;}
    #wordspacing3 {word-spacing:-1em;}

    <!-- HTML -->
    <p>Standard avstånd är 0.25em</p>
    <p id="wordspacing1">1 em avstånd</p>
    <p id="wordspacing2">2 em avstånd</p>
    <p id="wordspacing3">-1 em avstånd</p>

Height och width

Med dessa egenskaper kan man sätta en bestämd höjd respektive bredd på ett element.

För lite mer responsiv hantering av innehållet i t ex en <div> så kan man även använda sig av min/max-height/width.

    /* CSS */
    .coloredClass { background-color: blue;}
    .narrowClass { width: 200px; background-color: red;}
    .shortClass { height: 20px; width: 120px; 
        background-color: green;}

    <!-- HTML -->
    <div class="coloredClass"> <p> Här är det obehindrat</p>
        <div class="narrowClass">
            <p>Här var det smalt.</p>
            <div class="shortClass">
                <p>Här trångt.</p>
            </div>
        </div>
    </div>

Float

Med float kan man formatera ett element så att den lägger sig till vänster eller höger om ett annat element. Används ofta för positionering av t ex bilder.

Float kan sättas till värdena left, right, none och inherit.

    /* CSS */
    #centertext { text-align:center; }
    #floatimg1 { float:left; }
    #floatimg1 { float:right; }

    <!-- HTML -->
    <p id="centertext">Här är en centrerad text</p>
    <img id="floatimg1" src="../images/scroll-gray.png"
        alt="vänster">
    <img id="floatimg2" src="../images/scroll-gray.png"
        alt="vänster">

Position

Med position har man större kontroll över positioneringen av sina element.

Position kommer med fem värden: static, relative, fixed, absolute och sticky.

Ett element har automatiskt värdet static om inget annat anges. Static följer det vanliga flödet på HTML-sidan och påverkas ej av egenskaperna för top, bottom, left och right.

Relative lägger sig på samma plats som static men man kan därefter flytta denna med ovan egenskaper.

Fixed ställs in med en fast plats i relation till viewporten.

Absolute positionerar ett element i relation till närmaste övergripande (ancestor) element. Den kan då lägga ett element över andra som finns i ett annat flöde. Finns det ingen positionerad ancestor så följer den istället dokumentets kropp.

Slutligen så ger sticky möjligheten att positionera ett element baserat på användarens scroll-placering. Den lägger sig som ett element med relativ positionering tills användarens scroll position når en given plats, då blir den istället fixed.

    /* CSS */
    .staticDiv { position: static; border: 3px solid #cccccc;}
    .relativeDiv { position:relative; left:50px; border: 
        3px solid #cccccc; }
    .fixedDiv { position:fixed; bottom:200px; right:50px; 
        width: 200px; border: 3px solid #cccccc;}
    .absoluteDiv {position:absolute; top:90px; right: 0; 
        width:200px; height:50px; border: 3px solid #cccccc }
    .stickyDiv { position: sticky; top: 0px; 
        background-color:green; border: 3px solid #cccccc}


     <!-- HTML -->
     <div class="stickyDiv"><p>Här är en sticky
        positionerad div</p></div>
     <p>Här är ovan är en sticky</p>
     <br><br><br><br>
     <p>Den absoluta diven syns uppe till höger.</p><br>
     <p>Den fixerade syns nere till höger</p><br>
     <p>Den statiska och relativa syns här nedan</p>
                 
     <div class="staticDiv"><p>Här är en statiskt
        positionerad div</p></div>

Border

Med egenskapen border kan man sätta en ram runt ett element, samt stilsätta ramen. Nedan ges några exempel.

    /* CSS */
    #dotted { border-style:dotted; border-width:thick;}
    #dashed { border-style:dashed; border-width:thin;}
    #solid { border-style:solid; border-color:green;}
    #double { border-style:double; border-radius: 50px;}
    #mixed {   border-top-style: dotted; border-right-style: 
        solid; border-bottom-style: dotted; 
        border-left-style: solid;}

    <!-- HTML -->
    <p id="dotted">Tjocka prickar till kanter</p>
    <p id="dashed">Tunna streck till kanter</p>
    <p id="solid">Solid, grön kant</p>
    <p id="double">Dubbla, rundade kanter</p>
    <p id="mixed">Den här har blandade kanter</p>

Padding

Egenskapen genererar utrymme runt ett elements innehåll, innanför definerade gränser.

    /* CSS */
    #paddedDiv1 { padding: 150px; border: 3px solid #cccccc; }
    #paddedDiv2 { padding: 50px; border: 3px solid #cccccc;
         background-color: green; }
    #paddedDiv3 { padding-left: 150px; border: 3px solid #cccccc; }

    <!-- HTML -->
    <div id="paddedDiv1">
        <p>Här är (extra) paddning på 150px</p>
    </div>
    <div id="paddedDiv2">
        <p>Här är (extra) paddning på 50px</p>
    </div>
    <div id="paddedDiv3">
        <p>Här är (extra) paddning på 150px, men bara från
        vänster</p>
    </div>

Här är (extra) paddning på 150px

Här är (extra) paddning på 50px och färgat

Margin

Egenskaper ser till att ett element håller ett visst avstånd från närliggande element.

Margin, precis som padding, kan sättas för specifika riktningar.

Till skillnad från paddning så skappar margin inte något extra utrymme.

    /* CSS */
    #marginLeft {margin-left:100px;}
    #marginUp {margin-top: 100px;
        background-color: green;}


    <!-- HTML -->
    <p id="marginLeft">Håller sig från vänster</p>
    <p id="marginUp">Håller sig uppifrån och är färgad </p>

Håller sig från vänster

Håller sig uppifrån och är färgad
