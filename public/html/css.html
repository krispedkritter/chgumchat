CSS


Hur använder man CSS?

Det finns tre sätt hur man kan infoga CSS stilisering på: extern, intern och inline.

Med extern CSS använder man sig utav en separat fil (style sheet) för att hantera all stilisering på en och samma plats. Samma .css-fil kan användas på hela webbsidan.

Intern CSS används om en specifik HTML-sida ska ha specifik stilsättning. Den deklareras som extern, i <head>

Inline stil deklareras unikt för ett enda element.

    <!-- Extern style sheet deklarerat i <head> -->
    <head>
        <link rel="stylesheet" type="text/css" 
            href="../mystyle.css">
    </head>

    <!-- Intern style i <head> -->
    <head>
        <style>
            /* CSS-kod här */
            h1 { color:red;}
        </style>
    </head>

    <!-- Inline CSS --<
    <h1 style="color:red;">
        Det här skulle bli en röd mening.</h1>


Selektorer

Det finns fem typer av grundselektorer som används för att välja element som skall stilsättas.

Dessa selektorer är av typerna: universala, typ, klass, id och attribut.

En universal selektor väljer alla element. Den kan begränsas till ett specifikt namnområde (namespace).

En typ-selektor används för att stilisera en typ av HTML-tagg över hela webbsidan. Detta kan te x vara <p> eller <h1>.

Klass- och id stilsätter alla element av specificerad klass respektive id.

Slutligen kan attribut-selektorn användas för att stilsätta de element som har ett givet attribut.

Selektorer kan grupperas tillsammans med ett kommatecken , samt kombineras i olika utföranden. Se denna länk för mer information.

    /* CSS */
    /* Universal selektor */
    * {}
    /* Typ-selektor */
    elementnamn {}
    /* Klass-selektor */
    .klassnamn {}
    /* ID-selektor */
    #idnamn {}
    /* Attribut-selektor */
    [autoplay] {}


Deklarationsblock

Efter att ha bestämt vilken selektor som ska användas för stilsättningen, så är det dags att koda lite sådant.

All stilsättning sker inom deklarationsblocket { ... } för en selektor.

I exemplet nedan lägger vi bla lite färg på ID-selektorn för id #message.

    /* CSS */
    #message {
        background-color:blue;
        color:red;
        font-weight:500;
    }

    <!-- HTML -->
    <p id=message>Hallå där!</p>

Hallå där!



Klasser .class och ID #id

Om man istället vill stilsätta flera element samtidigt så gör man oftast det med hjälp av antingen klasser eller id.

Med en klass kan du använda samma stilsättning på flera element så som <div>-taggar, en för en, inkapslade i varandra eller vad man nu kan hitta på.

ID används generellt för att stilsätta enstaka element, en för en.

    <!-- HTML -->
    <div class="fistClass">
        <p>Första klassen är grå.</p>
        <div class="secondClass">
            <p>Andra klassen är röd.</p>
            <p class="firstClass">Denna och nästa rad 
                deklarerades som första inne i andra och är 
                därför grå.</p>
            <p class="firstClass">Notera att vänstersidan är
                 röd pga marginal i extern stilsättning.</p>
            <p class="firstClass" id="uniqueRow">Den här 
                raden är unikt orange mha #id.</p>
            <p>...och sista raden är röd.</p>
        </div>
    </div>

    /* CSS */
    .firstClass {
        background-color:grey;
    }
    .secondClass {
        background-color:red;
    }
    #uniqueRow { 
        background-color:orange;
    }

Första klassen är grå.

Andra klassen är röd.

Denna och nästa rad deklarerades som första inne i andra och är därför grå.

Notera att vänstersidan är röd pga marginal i extern stilsättning.

Den här raden är unikt orange mha #id.

...och sista raden är röd.



Prioriteringsregler och !important

När man arbetar med CSS och stilsättning så är det viktig att känna till de prioriteringsregler som finns, som avgör vilken stilsättning ett element till slut får om den får stilsättning från olika håll.

Först och främst så är det den sist deklarerade selektorn, om det finns flera stilsättningar på en och samma selektor, som används:

    /* CSS */
    p{ color: green; }
    p{ color: red ;}
    /* stycken p blir röda */

Har man flera olika typer av stilsättning så går CSS efter en inbyggd specifikations-faktor som är numrerad 1-100. Det högsta värdet används.

#id selektorn har värdet 100

.class selektorn har värdet 10

tag selektorn har värdet 1

    /* CSS */
    #messageId { color:red;}
    .messageClass {color:blue;}
    p{ color: green;}

Röd text då ID har prio 100.

Vill man trumfa den inbyggda prioriteringsregeln så kan man använda sig av regeln !important.

!important kör över ALL tidigare stilisering för det elementet.

    /* CSS */
    #messageId2 { color:red;}
    #messageId2 { color:green;}
    .messageClass2 {color:blue !important;}

Blå text då !important kör över #id
